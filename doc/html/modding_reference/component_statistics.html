Statistics component¶

A statistics component collects statistics for an entity, such as ships killed, shots fired, and so on. While it can be declared in YAML, it only makes sense (and is used) for the player ship, statistics of which are used for player scores. It might be used in future for RPG elements.

Example:

statistics:

Currently, this component has no parameters.
