struct PNGEncoder;

Encodes image data to PNG format.

@property void level(const ubyte level);

Set compression level. Can't be greater than 9.

@property void compression(const CompressionStrategy compression);

Set Zlib compression strategy.

@property void filter(const PNGFilter filter);

Set PNG filter strategy.

@property void compressText(const bool compress);

Compress text data? (on by default)

ubyte[] encode(const PNGInfo info, const ubyte[] source);

Encode image data to PNG format.

Only 24bit RGB and 32bit RGBA formats are supported at the moment.

Parameters:
PNGInfo info PNG information to encode.
ubyte[] source Image data to encode. Must correspond to width, height and color type in info.
Returns:
Manually allocated array with encoded PNG data. Must be manually freed.
Throws:
PNGException on failure.
void addUint(T)(ref T buffer, const const(uint) i);

Append a uint to specified ubyte buffer (vector or array).

Parameters:
buffer Buffer to append to.
i uint to append.
