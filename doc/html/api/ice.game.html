class Game;

Class managing a single game between players.

enum GamePhase;

Game phases.

Playing

Gameplay phase.

PreOver

Phase during the game over effect, before the score screen shows up.

Over

Game over - we're not playing but we still aren't out of the game (e.g. score screen).
